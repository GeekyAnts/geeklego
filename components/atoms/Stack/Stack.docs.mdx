## Accessibility

- `<div>` by default — no ARIA role of its own
- Use `as` prop to render semantic elements (`nav`, `ul`, `section`) when needed
- Add `aria-label` when using a landmark element (e.g. `as="nav"`)
- Stack is a transparent layout container — screen readers see only its children
- No keyboard interaction — layout only
