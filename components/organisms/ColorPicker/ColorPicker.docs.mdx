## Accessibility

- Root element is `<div role="group">` labelled by a screen-reader-only heading
- Spectrum is a 2D `role="slider"` with `aria-valuemin`, `aria-valuemax`, `aria-valuenow`, and `aria-label`
- Hue and alpha sliders use `role="slider"` with full `aria-value*` attributes
- Format buttons use `aria-pressed` to indicate the active format (HEX/RGB/HSL)
- Preset swatches use `aria-pressed` for the selected color with required `aria-label`
- Copy button `aria-label` switches between copy label and "Copied!" confirmation

### Keyboard Interaction

| Key | Action |
|-----|--------|
| Tab | Moves focus between all focusable controls |
| Arrow keys | Adjust spectrum saturation (left/right) or brightness (up/down) by 1% |
| Shift + Arrow keys | Adjust spectrum by 10% |
| Arrow keys | Adjust hue/alpha slider value by 1 |
| Shift + Arrow keys | Adjust hue/alpha slider by 10 |
| Home / End | Jump to min/max on spectrum or slider |
| Enter / Space | Switch format, select preset color, or copy hex |
