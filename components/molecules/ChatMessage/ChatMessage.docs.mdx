## Accessibility

- `<li>` element — must be placed inside a `<ul>` or `<ol>` parent
- Avatar receives `alt=[senderName]` — providing the sender's name to screen readers
- Sender name span is `aria-hidden="true"` — same information conveyed via avatar `alt`
- When `showName` is `false`, avatar receives `aria-hidden={true}` to avoid double-announcement
- Bubble content read by screen readers in natural DOM order (name, bubble text, timestamp)
- No keyboard interaction — display-only component
