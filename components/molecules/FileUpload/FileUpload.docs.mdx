## Accessibility

- `<label>` element wraps the drop zone — clicking anywhere in the zone activates the hidden `<input type="file">`
- Hidden `<input type="file">` receives `aria-label` from `i18nStrings.inputLabel` (default: "Upload files")
- `aria-disabled` set on both the `<label>` and `<input>` when `disabled` is true
- `aria-invalid` and `aria-describedby` applied to the input via `getErrorFieldProps()` when `error` and `errorMessage` are set
- File list renders as `<ul aria-label="Selected files">` with each file as `<li>`
- File meta paragraph uses `aria-live="polite" aria-atomic="true"` to announce status changes (Done, Uploading, error)
- Zone error message uses `role="alert"` for immediate announcement
- Remove buttons include the filename: `aria-label="Remove file: filename.pdf"`

### Keyboard Interaction

| Key | Context | Action |
|-----|---------|--------|
| Tab | Component | Moves focus to the hidden file input |
| Enter / Space | Hidden input focused | Opens the native file picker |
| Tab | File list | Moves through remove buttons |
| Enter / Space | Remove button focused | Removes the file |
