### BreadcrumbItemData

| Field | Type | Description |
|-------|------|-------------|
| `label` | `string` | The visible label text. Required. |
| `href` | `string` | URL this item links to. Omit for the current page. |
| `current` | `boolean` | Explicitly marks as current page. Auto-applied to the last item. |
| `disabled` | `boolean` | Non-interactive, visually muted. |
| `icon` | `ReactNode` | Optional icon node before the label. |

## Accessibility

- Wrapped in `<nav aria-label="Breadcrumb">` — announces as a navigation landmark
- Items are rendered as `<ol>` / `<li>` — communicates ordered sequence to screen readers
- Separator icons are `aria-hidden="true"` — not read by screen readers
- The last item receives `aria-current="page"` via the underlying BreadcrumbItem atom
- Disabled items are non-interactive and visually muted
- Schema.org `BreadcrumbList` and `ListItem` Microdata available via the `schema` prop

### Keyboard Interaction

| Key | Action |
|-----|--------|
| Tab | Moves focus between breadcrumb links |
| Enter | Navigates to the focused link |
