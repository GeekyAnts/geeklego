## Accessibility

- `<figure>` wraps the media unit and receives `aria-label` and optional Schema.org itemscope
- `<video>` element also receives `aria-label` so screen readers can identify it
- `<figcaption>` renders only when `caption` is provided
- `<track kind="captions">` required for WCAG 1.2.2 (Captions — Prerecorded)
- Native `<video controls>` is fully keyboard-navigable
- For ambient/decorative video, provide an explicit `aria-label` stating no information is conveyed

### Keyboard Interaction

| Key | Action |
|-----|--------|
| Tab | Moves focus into the video controls |
| Space | Play / pause |
| Arrow keys | Seek forward/backward (browser-dependent) |
| M | Toggle mute (browser-dependent) |
| F | Toggle fullscreen (browser-dependent) |
