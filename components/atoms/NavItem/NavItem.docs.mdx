## Accessibility

- Renders as a `<li>` inside a `role="tree"` container — must always be wrapped in `<ul role="tree">`
- When `href` is provided, renders as `<a>` — announces as a link; `isActive` sets `aria-current="page"`
- When no `href`, renders as `<button>` — announces as a button
- `isExpandable` adds `aria-expanded` and `aria-controls` on the trigger; sub-items have `role="group"`
- `disabled` sets `aria-disabled="true"` on the inner element; the item is visually muted but may remain focusable
- Icons are `aria-hidden="true"` — decorative, not announced by screen readers
- Supports roving tabindex via `innerTabIndex` for arrow-key navigation in parent containers (e.g. Sidebar)

### Keyboard Interaction

| Key | Action |
|-----|--------|
| Tab | Moves focus to the item (or use arrow keys with roving tabindex) |
| Enter | Activates the link or toggles expansion |
| Space | Toggles expansion on expandable items |
| Arrow Down / Up | Navigates between items when roving tabindex is active |
