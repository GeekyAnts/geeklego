## Accessibility

- Renders as a native `<input type="range">` — all keyboard interaction is browser-native
- `label` provides the accessible name via `aria-labelledby`; without `label`, pass `aria-label`
- `aria-valuemin`, `aria-valuemax`, `aria-valuenow`, and `aria-valuetext` are set automatically
- The visual value display (when `showValue`) is `aria-hidden` — screen readers get the value from `aria-valuenow`
- `disabled` sets both HTML `disabled` and `aria-disabled`

### Keyboard Interaction

| Key | Action |
|-----|--------|
| Tab | Moves focus to the slider |
| Arrow Right / Up | Increases value by step |
| Arrow Left / Down | Decreases value by step |
| Home | Sets value to minimum |
| End | Sets value to maximum |
| Page Up | Increases by 10× step |
| Page Down | Decreases by 10× step |
