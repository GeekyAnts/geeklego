## Accessibility

- Renders as a `<button role="switch">` — announces as "switch, on" or "switch, off" to screen readers
- `children` is used as the accessible name via `aria-labelledby`
- When no `children`: supply `aria-label` directly (e.g. `aria-label="Toggle dark mode"`)
- `description` is linked via `aria-describedby` — read after the label and role
- `disabled` sets both HTML `disabled` and `aria-disabled` attributes
- Focus ring applied on keyboard navigation via `focus-visible`

### Keyboard Interaction

| Key | Action |
|-----|--------|
| Tab | Moves focus to the switch |
| Space | Toggles the switch on/off |
| Enter | Toggles the switch on/off |
