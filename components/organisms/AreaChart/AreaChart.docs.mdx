## Accessibility

- SVG chart has `role="img"` with a descriptive `aria-label` derived from the title
- A visually hidden `<table>` provides a full data fallback for screen readers, with column headers for each series
- Info button uses `title` attribute for accessible labelling
- Period selector has an `aria-label` for screen reader context
- `aria-busy` and `aria-live` attributes applied during loading state via `getLoadingProps`
- Legend swatches are `aria-hidden="true"` to avoid redundant announcements
- Tooltip dot indicators use `aria-hidden="true"`
- Tooltip container has `role="tooltip"`

### Keyboard Interaction

| Key | Action |
|-----|--------|
| Tab | Moves focus to the info button (if present), then to the period selector |
| Enter / Space | Activates the info button or opens the period selector |
| Arrow Up / Down | Navigates period selector options when open |
| Escape | Closes the period selector dropdown |
