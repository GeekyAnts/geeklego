## Accessibility

- Wrapped in `<nav aria-label="Pagination">` — announced as a navigation landmark
- Current page button has `aria-current="page"`
- Previous and next buttons are icon-only with accessible labels via `i18nStrings`
- A screen-reader-only `aria-live="polite"` region announces the current page whenever it changes
- Ellipsis elements are `aria-hidden="true"` — not read by screen readers
- Disabled buttons use `aria-disabled` and cannot receive focus

### Keyboard Interaction

| Key | Action |
|-----|--------|
| Tab | Moves focus between prev, page number, and next buttons |
| Enter | Activates the focused page button |
| Space | Activates the focused page button |
