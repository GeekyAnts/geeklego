### DropdownMenuItemDef

| Field | Type | Description |
|-------|------|-------------|
| `id` | `string` | Unique stable identifier. Required. |
| `label` | `string` | Visible item label. Required. |
| `icon` | `ReactNode` | Leading icon node. |
| `shortcut` | `string` | Keyboard shortcut label rendered on the trailing edge (e.g. `"⌘C"`). |
| `href` | `string` | When provided, renders the item as an anchor instead of a button. |
| `target` | `string` | Link target attribute (e.g. `"_blank"`). |
| `rel` | `string` | Link rel attribute. |
| `disabled` | `boolean` | Non-interactive and visually muted. |
| `destructive` | `boolean` | Renders the item in the error/destructive colour. |

| `items` | `DropdownMenuItemDef[]` | Items belonging to this group. Required. |

## Accessibility

- Menu panel has `role="menu"` — screen readers announce it as a menu
- Each item is `role="menuitem"` — navigable with arrow keys
- Trigger receives `aria-haspopup="menu"`, `aria-expanded`, and `aria-controls` injected automatically
- Escape key closes the menu and returns focus to the trigger
- Click outside closes the menu
- Disabled items have `aria-disabled="true"` and are non-interactive
- Separator is `role="separator"` and `aria-hidden="true"`

### Keyboard Interaction

| Key | Action |
|-----|--------|
| Enter / Space | Opens the menu (on trigger) or activates the focused item |
| Arrow Down | Opens the menu or moves focus to the next item |
| Arrow Up | Moves focus to the previous item |
| Escape | Closes the menu and returns focus to the trigger |
| Tab | Closes the menu |
| Home | Moves focus to the first item |
| End | Moves focus to the last item |
