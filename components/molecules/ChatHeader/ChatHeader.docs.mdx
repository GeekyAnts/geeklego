## Accessibility

- Semantic `<header>` element — landmark scoped to parent `<section>`
- Avatar `alt` set to the `name` prop — provides accessible participant name
- Status dot uses `aria-hidden="true"` — status conveyed by text label, not color alone
- Visible status label follows the dot — meaning is not color-only
- Action buttons must have accessible names via the `iconOnly` prop on `Button`

### Keyboard Interaction

| Key | Action |
|-----|--------|
| Tab | Moves focus through action buttons (if present) |
| Enter | Activates the focused action button |
| Space | Activates the focused action button |
