## Accessibility

- `role="status"` marks the element as a live region status container
- `aria-live="polite"` — screen reader announces politely when the component mounts
- Dots use `aria-hidden="true"` — hidden from the accessibility tree
- sr-only text: "[name] is typing…" when `name` provided, or "Typing…" when omitted
- `typingLabel` i18n string allows overriding the "is typing…" portion for non-English locales
- No keyboard interactions — purely visual/auditory indicator
