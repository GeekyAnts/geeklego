## Accessibility

- Wrapped in `role="search"` — announced as a search landmark by screen readers
- `<label>` is linked to the input via `htmlFor`/`id` — `id` is auto-generated with `useId()` when omitted
- Clear button has a localised `aria-label` via `i18nStrings`
- Error message uses `role="alert"` — announced immediately when it appears
- Loading state sets `aria-busy="true"` on the input
- Schema.org `SearchAction` JSON-LD injected via `<script type="application/ld+json">` when `schema` and `searchUrl` are provided

### Keyboard Interaction

| Key | Action |
|-----|--------|
| Tab | Moves focus to the search input, clear button, or search button |
| Enter | Submits the search query |
| Escape | Clears the input when focused |
