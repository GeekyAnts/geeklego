## Accessibility

- Renders as `<a>` when `href` is provided (native link semantics)
- Renders as `<div role="button" tabIndex={0}>` when `onClick` is provided without `href`
- Renders as plain `<div>` when non-interactive
- `aria-selected` set when `selected` is true
- `aria-disabled` set when `disabled` is true
- `aria-busy` set when `loading` is true
- Media slot rendered with `aria-hidden="true"` (decorative)
- Keyboard: Enter and Space trigger `onClick` on `div[role=button]`
- Disabled items are removed from the tab order

### Keyboard Interaction

| Key | Action |
|-----|--------|
| Tab | Moves focus to the item |
| Enter | Activates the item (click or follow href) |
| Space | Activates the item (click only, not links) |
