## Accessibility

- Interactive mode renders as `<fieldset>` + `<legend>` with radio inputs — full radio-group semantics
- Screen readers announce each star as "N out of M stars, radio button"
- `label` provides the group accessible name via `<legend>` (visually hidden by default)
- Read-only mode renders `<span role="img" aria-label="N out of M stars">` — purely informational
- `disabled` sets `fieldset disabled` which disables all contained radio inputs at the browser level
- `schema` adds Schema.org `AggregateRating` Microdata when `readOnly={true}`

### Keyboard Interaction

| Key | Action |
|-----|--------|
| Tab | Moves focus into the rating group |
| Arrow Left / Right | Moves selection between stars |
| Space / Enter | Selects the focused star |
