## Accessibility

- Panel has `role="dialog"` with `aria-modal="true"` — screen readers announce it as a dialog
- When `title` is provided, `aria-labelledby` links the panel to the title heading
- `aria-hidden` toggles on the panel to hide it from assistive technology when closed
- Trigger receives `aria-haspopup="dialog"`, `aria-expanded`, and `aria-controls` automatically
- Focus is trapped inside the panel while open — Tab cycles through all interactive elements
- Escape closes the panel and returns focus to the trigger
- Click outside the component closes the panel
- Close button has a localised `aria-label` via `i18nStrings.closeLabel`

### Keyboard Interaction

| Key | Action |
|-----|--------|
| Enter / Space | Opens the popover (on trigger) |
| Tab | Cycles focus through interactive elements inside the panel |
| Shift + Tab | Reverses focus cycle inside the panel |
| Escape | Closes the panel and returns focus to the trigger |
