## Accessibility

**Semantic elements:** `<ul>` (bullet/check/dot/none), `<ol>` (ordered), `<dl>` (description)

| Element | Implicit role |
|---|---|
| `<ul>` | `list` |
| `<ol>` | `list` |
| `<li>` | `listitem` |
| `<dt>` | `term` |
| `<dd>` | `definition` |

- `aria-disabled="true"` applied to `<li>` when `disabled={true}`
- Check and dot icons carry `aria-hidden="true"` — text content carries all meaning
- Screen readers announce item count and position (e.g. "list, 4 items", "item 1 of 4")
- `<dl>` announces term/definition pairs in supported screen readers

### Keyboard Interaction

| Key | Behaviour |
|-----|-----------|
| Tab | Moves focus to the next interactive element inside a list item |
| Enter / Space | Activates any button or link within the focused item |
