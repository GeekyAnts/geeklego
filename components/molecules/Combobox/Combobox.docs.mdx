### ComboboxOption

| Field | Type | Description |
|-------|------|-------------|
| `id` | `string` | Unique identifier — used as React key and selection value. Required. |
| `label` | `string` | Visible label shown in the input and option list. Required. |
| `description` | `string` | Optional secondary text rendered below the label. |
| `disabled` | `boolean` | Option is visible but cannot be selected. |
| `group` | `string` | Group name — options with the same value are visually grouped with a heading. |

## Accessibility

- Trigger input has `role="combobox"` with `aria-expanded`, `aria-controls`, `aria-autocomplete="list"`, and `aria-haspopup="listbox"`
- Active option is communicated via `aria-activedescendant` — no focus moves to the list
- Each option has `role="option"` with `aria-selected` and `aria-disabled` when applicable
- Option list has `role="listbox"` with a localised `aria-label`
- Groups use `role="group"` with `aria-label` matching the group name
- Loading and empty states are announced as `role="option"` with `aria-disabled="true"`
- Clear button has a localised `aria-label` via `i18nStrings.clearLabel`
- Escape dismisses the panel without selecting

### Keyboard Interaction

| Key | Action |
|-----|--------|
| Arrow Down | Opens the panel or moves focus to the next option |
| Arrow Up | Moves focus to the previous option |
| Home | Moves focus to the first option |
| End | Moves focus to the last option |
| Enter | Selects the focused option |
| Escape | Closes the panel without selecting |
| Tab | Closes the panel and moves focus to the next element |
