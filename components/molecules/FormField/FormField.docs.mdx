## Accessibility

- Renders a `<label>` via the Label atom with native `htmlFor` association to the form control
- Hint text receives a deterministic `id` of `[htmlFor]-hint` for `aria-describedby` wiring
- Error message receives `role="alert"` and `id` of `[htmlFor]-error` — announced immediately by screen readers when it appears
- Required state shows a decorative red asterisk (`aria-hidden`) plus sr-only "(required)" text
- Consumer is responsible for wiring `aria-describedby` on the control to the hint or error `id`

### Keyboard Interaction

| Key | Action |
|-----|--------|
| Tab | Moves focus to the form control inside the field |
| Click on label | Focuses the associated control (native label behavior) |
