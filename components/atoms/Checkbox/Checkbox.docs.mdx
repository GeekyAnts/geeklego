## Accessibility

- Renders a native `<input type="checkbox">` — full browser accessibility support
- `aria-checked="mixed"` set when `indeterminate` is true
- `aria-invalid="true"` set when `error` is true
- `aria-required="true"` set when `required` is true
- `aria-disabled="true"` set when `disabled` is true
- Label is associated to the input via a wrapping `<label>` element
- When no `children` label is provided, supply `aria-label` to describe the checkbox
- `focus-visible` focus ring for keyboard navigation

### Keyboard Interaction

| Key | Action |
|-----|--------|
| Tab | Moves focus to the checkbox |
| Space | Toggles the checked state |
