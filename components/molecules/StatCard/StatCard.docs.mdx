## Accessibility

- `<article>` element for a self-contained content unit
- `<dl>` + `<dt>` (label) + `<dd>` (value + delta) — definition list semantics
- Screen-reader-only trend prefix ("Trending up: " / "Trending down: ") prepended to the delta badge content
- Icon wrapper has `aria-hidden="true"` — decorative
- Loading state sets `aria-busy="true"` with `aria-label` on the article element
