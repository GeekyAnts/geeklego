## Accessibility

Implements **WCAG 2.1 SC 2.4.1 — Bypass Blocks (Level A)**. Keyboard-only users can Tab to the skip link, which slides into view, then press Enter to move focus to the target landmark.

- Semantic `<a href="...">` — a native hyperlink activated with Enter
- No `aria-label` needed — the visible label text is the accessible name
- Landmark targets must use `tabIndex={-1}` to receive programmatic focus

### Keyboard Interaction

| Key | Action |
|-----|--------|
| Tab | Move focus to the skip link (link slides into view) |
| Enter | Follow the href to the target landmark; focus moves to the target |
| Tab (again) | Move focus past the skip link to the next focusable element |
