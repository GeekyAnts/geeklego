## Accessibility

- Renders as `role="progressbar"` with `aria-valuemin`, `aria-valuemax`, `aria-valuenow`, and `aria-valuetext`
- `label` is used as `aria-label` — always provide a descriptive label
- Indeterminate state (no `value`) sets `aria-busy="true"` and omits `aria-valuenow`
- For page-level loading, consider placing in a `role="status"` live region so changes are announced
- Purely visual — no keyboard interaction
