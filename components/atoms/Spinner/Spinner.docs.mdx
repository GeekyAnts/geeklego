## Accessibility

- Renders with `role="status"` — a polite ARIA live region that announces the label when the spinner appears
- The `label` prop is read by screen readers via a visually-hidden `<span>`; defaults to `"Loading…"`
- Pass a specific `label` for context: `label="Uploading file, please wait"`
- `aria-label` attribute overrides the `label` prop when more dynamic text is needed
- For container-level loading, pair with `aria-busy="true"` on the parent element
- Animation collapses to a static arc when `prefers-reduced-motion: reduce` is set
- No interactive role — keyboard users do not focus the spinner
