## Accessibility

- Renders a native `<input>` — full browser and assistive technology support
- Always associate with a `<label>` via `htmlFor` + `id`, or pass `aria-label` for unlabeled inputs
- `aria-invalid="true"` set when `error` is true — pair with `aria-describedby` pointing to an error message
- `aria-busy="true"` set when `isLoading` is true
- Icon slots are `aria-hidden="true"` — they are decorative
- `disabled` sets both the HTML `disabled` attribute and `aria-disabled`

### Keyboard Interaction

| Key | Action |
|-----|--------|
| Tab | Moves focus to the input |
| Any key | Types into the field |
