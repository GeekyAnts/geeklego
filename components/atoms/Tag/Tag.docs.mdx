## Accessibility

- **Semantic element:** `<span>` (static or removable), `<a>` (linked)
- `aria-label` for additional context when needed
- `aria-hidden="true"` on leading icon wrapper — decorative
- Remove button has `aria-label` (default: "Remove", i18n-configurable)
- Linked tags are focusable via Tab and activated with Enter
- Static tags without interaction are not focusable
- All variant and colour combinations meet WCAG 2.2 AA minimum contrast ratio

### Keyboard Interaction

| Key | Action |
|-----|--------|
| Tab | Moves focus to a linked tag or its remove button |
| Enter | Activates link navigation or remove action |
| Space | Activates remove button |
