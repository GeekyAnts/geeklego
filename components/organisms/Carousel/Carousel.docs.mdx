## Accessibility

- Semantic `<section>` landmark with `aria-roledescription="carousel"` and `aria-label` from the required `label` prop
- Each slide is a `<li>` with `role="group"` and `aria-roledescription="slide"`, labelled "X of N"
- Off-screen slides have `aria-hidden="true"` to prevent screen reader announcement
- A polite live region announces the current slide number on every navigation (`aria-live="polite" aria-atomic="true"`)
- Track `<ul>` has `aria-live="off"` to suppress individual child announcements
- Previous / Next buttons use `aria-label` from i18n strings
- Dot indicators use `aria-label="Go to slide N"` and `aria-current="true"` on the active dot
- Autoplay toggle uses `aria-pressed` to expose play/pause state
- `aria-busy="true"` applied via `getLoadingProps()` when `loading` is true
