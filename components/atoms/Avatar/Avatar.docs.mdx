## Accessibility

- `role="img"` on non-image variants (initials, icon, fallback)
- `alt` text provides the accessible name for image variants
- `aria-label` defaults to `"User avatar"` for initials, icon, and fallback variants
- When `iconOnly`, the icon is `aria-hidden="true"` and the text children serve as the label
- `loading` renders a skeleton placeholder with `aria-busy="true"`
- Decorative avatars can pass `alt=""` to silence screen reader announcements
- No keyboard interaction — Avatar is a display-only component
