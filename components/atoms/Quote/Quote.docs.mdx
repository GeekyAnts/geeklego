## Accessibility

- Renders as `<figure>` + `<blockquote>` + `<figcaption>` — semantic, screen-reader-friendly structure
- When `attribution` is provided, `<figcaption>` associates the attribution with the quote
- `source` renders inside a `<cite>` element — communicates semantic citation meaning
- `sourceUrl` makes the source text a link; `cite` sets `blockquote[cite]` for AT context
- Decorative elements (quotation marks in pullquote) are `aria-hidden="true"`
- The `sourceUrl` link has a visible focus ring for keyboard navigation
- No interactive elements unless `sourceUrl` is provided

### Keyboard Interaction

| Key | Action |
|-----|--------|
| Tab | Moves focus to the source link (if sourceUrl is provided) |
| Enter | Activates the source link |
