## Accessibility

- `role="group"` on the container — screen readers announce the group name on entry
- `aria-label` is required to give the group a meaningful name; use `aria-labelledby` when a visible heading already labels the group
- Each button inside retains its own accessible name independently
- Disabled buttons are announced as dimmed by VoiceOver
- No arrow-key navigation — buttons are individually Tab-focusable

### Keyboard Interaction

| Key | Action |
|-----|--------|
| Tab | Moves focus between buttons in the group |
| Enter | Activates the focused button |
| Space | Activates the focused button |
