## Accessibility

- Renders as `<button role="button">` with `aria-pressed` set to `"true"` or `"false"`
- Screen readers announce: `"[label], toggle button, pressed"` or `"[label], toggle button, not pressed"`
- `children` provides the accessible name; for icon-only usage, pass `aria-label`
- Wrap toolbar groups of toggles in `<div role="toolbar" aria-label="...">` for grouped context
- `disabled` sets both HTML `disabled` and `aria-disabled`
- Focus ring applied on keyboard navigation via `focus-visible`

### Keyboard Interaction

| Key | Action |
|-----|--------|
| Tab | Moves focus to the toggle |
| Space | Toggles pressed state |
| Enter | Toggles pressed state |
