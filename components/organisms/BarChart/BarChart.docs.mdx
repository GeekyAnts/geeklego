## Accessibility

- Bar chart container uses `role="img"` with an `aria-label` describing the chart title
- A visually hidden `<table>` provides a full data fallback for screen readers, with columns for segment name, value, and percentage share
- Info button uses `title` attribute for its accessible name
- Period selector has `aria-label` for screen reader context
- Colour legend swatches are marked `aria-hidden="true"` so they do not announce to screen readers
- Loading state uses `aria-busy` and `aria-live` attributes via `getLoadingProps`
- Tooltip uses `role="tooltip"` for proper semantics
- Up to 7 distinct series colours ensure visual differentiation

### Keyboard Interaction

| Key | Action |
|-----|--------|
| Tab | Moves focus to the info button, then the period selector |
| Enter / Space | Activates the info button or opens the period selector dropdown |
| ArrowDown / ArrowUp | Navigates period selector options when open |
