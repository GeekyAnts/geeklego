## Accessibility

- `role="status"` for `info` and `success` variants — polite live region, screen reader waits for a reading pause before announcing
- `role="alert"` for `error` and `warning` variants — assertive live region, interrupts the screen reader immediately
- Dismiss button is icon-only with an accessible `aria-label` via `i18nStrings.dismissLabel`
- Status icon and progress bar are `aria-hidden="true"` — decorative, not read by screen readers
- When `dismissible={false}`, no interactive element is present and the toast has no Tab stop

### Keyboard Interaction

| Key | Action |
|-----|--------|
| Tab | Moves focus to the dismiss button (when present) |
| Enter | Activates the dismiss button |
| Space | Activates the dismiss button |
