## Sizes

Left/right drawers use `size` to control panel width; top/bottom drawers use it to control panel height. `full` fills the entire viewport with no border-radius.

## Accessibility

**Semantic element:** `<div role="dialog" aria-modal="true">` rendered in a portal on `document.body`.

**ARIA attributes:**

| Attribute | Value | Condition |
|---|---|---|
| `role` | `"dialog"` | Always |
| `aria-modal` | `"true"` | Always |
| `aria-labelledby` | Points to the `<h2>` title `id` | When `title` is provided |
| `aria-label` | `i18nStrings.drawerLabel` | When `title` is omitted |
| `aria-busy` | `"true"` | When `loading={true}` |

**Focus management:**
- Focus is trapped inside the panel while open via `useFocusTrap`.
- Escape dismissal is handled by `useEscapeDismiss`.
- On close, focus returns to the element that triggered the drawer.
- The backdrop carries `aria-hidden="true"` — it is a visual affordance only.

### Keyboard Interaction

| Key | Action |
|-----|--------|
| Tab | Moves focus to the next focusable element within the panel (trapped). |
| Shift + Tab | Moves focus to the previous focusable element within the panel. |
| Escape | Closes the drawer and returns focus to the trigger element. |
