## Accessibility

- Semantic `<button type="button">` element
- `aria-label` is required — describes the color to screen readers
- `aria-pressed` set to `true` when `selected`
- `aria-disabled` and `disabled` set when `disabled`
- `focus-visible` focus ring for keyboard navigation
- All sizes meet WCAG 2.5.8 minimum 24×24px touch target

### Keyboard Interaction

| Key | Action |
|-----|--------|
| Tab | Moves focus to the swatch |
| Enter | Activates the swatch |
| Space | Activates the swatch |
