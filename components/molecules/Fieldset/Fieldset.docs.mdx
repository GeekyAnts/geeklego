## Accessibility

- Rendered as a native `<fieldset>` with `<legend>` — screen readers announce the group name when entering the group
- `disabled` on the `<fieldset>` natively propagates to all descendant controls — no JavaScript required
- Error message uses `role="alert"` — announced immediately by screen readers when it appears
- Hint text is linked via `aria-describedby` on the fieldset
- Required indicator uses `aria-hidden` on the visual asterisk with a screen-reader-only text fallback

### Keyboard Interaction

| Key | Action |
|-----|--------|
| Tab | Moves focus between controls inside the group |
| Space | Toggles checkboxes or selects the focused radio |
| Arrow keys | Moves selection between radio buttons in the group |
