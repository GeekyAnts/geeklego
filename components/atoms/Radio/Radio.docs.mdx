## Accessibility

- Renders as a native `<input type="radio">` — full browser and assistive technology support
- Group radios with the same `name` attribute — browser handles mutual exclusion automatically
- Wrap groups in `<fieldset>` + `<legend>` for accessible group labeling
- `children` provides the visible label; without it, pass `aria-label` for unlabeled radios
- `error` shifts the border color; pair with `aria-describedby` pointing to an error message
- `disabled` sets the HTML `disabled` attribute — removes the input from tab order

### Keyboard Interaction

| Key | Action |
|-----|--------|
| Tab | Moves focus into the radio group |
| Arrow Up / Down / Left / Right | Moves selection within the group |
| Space | Selects the focused radio |
