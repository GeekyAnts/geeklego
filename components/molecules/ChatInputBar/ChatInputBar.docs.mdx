## Accessibility

- Root is a `<form>` — implicit form landmark semantics
- Attachment button is icon-only with `aria-label` (default: "Attach file") resolved via i18n
- Send button is icon-only with `aria-label` (default: "Send message") resolved via i18n
- When textarea is empty, send button is `disabled` with `aria-disabled="true"` — preventing keyboard activation
- Textarea carries `aria-label` matching the send button label for screen reader context
- `Ctrl/Cmd + Enter` shortcut is additive — standard form submit via `Enter` remains intact

### Keyboard Interaction

| Key | Action |
|-----|--------|
| Enter | Submits the message when the textarea has content |
| Ctrl+Enter / Cmd+Enter | Submits the message from inside the textarea |
| Tab | Moves focus between the attachment button, textarea, and send button |
