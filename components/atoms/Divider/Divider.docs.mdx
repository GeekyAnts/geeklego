## Accessibility

- Renders as `<hr role="separator">` with `aria-orientation` matching the `orientation` prop
- Screen readers announce it as a thematic break between sections
- Purely decorative dividers within already-separated regions can pass `aria-hidden="true"`
- No keyboard interaction — Divider is a display-only component
