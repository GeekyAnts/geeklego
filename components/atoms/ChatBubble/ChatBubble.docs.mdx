## Accessibility

- `<div>` element — no native interactive semantics, no keyboard interaction
- Timestamp rendered as `<time>` element for semantic date/time meaning
- Reading order conveyed by DOM order — use appropriate ordering in the parent container
- Variant communicated by background colour AND alignment AND border treatment — not colour alone
