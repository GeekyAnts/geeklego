## Accessibility

- `role="alert"` (assertive) for `error` and `warning` — screen reader interrupts current speech immediately
- `role="status"` (polite) for `info` and `success` — screen reader waits for a pause before announcing
- Leading icon wrapper is `aria-hidden` — colour is not the sole status cue (icon provides redundancy)
- Dismiss button has a localised `aria-label` via `i18nStrings.dismissLabel`
- Focus ring and 44×44px touch target on the dismiss button
- `actions` slot items are individually keyboard-reachable

### Keyboard Interaction

| Key | Action |
|-----|--------|
| Tab | Moves focus to the dismiss button or action items |
| Enter | Activates the focused button |
| Space | Activates the focused button |
