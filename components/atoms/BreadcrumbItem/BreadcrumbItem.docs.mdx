## Accessibility

- Link items render as `<a>` — focusable, supports Enter key activation
- `current` items render as non-interactive `<span>` with `aria-current="page"`
- `disabled` items are non-interactive with `aria-disabled="true"` and visually muted
- Icon nodes are `aria-hidden="true"` — the text label carries full meaning
- Wrap multiple `BreadcrumbItem` components in an `<ol>` element for proper list semantics
- The parent `<ol>` should be inside a `<nav aria-label="Breadcrumb">` landmark

### Keyboard Interaction

| Key | Action |
|-----|--------|
| Tab | Moves focus to the next link item |
| Enter | Activates the link |
